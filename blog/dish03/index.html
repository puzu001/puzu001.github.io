5道養生素食料理食譜

By Puzumei 23 July 2024

5道養生素食食譜

1. 素食椰奶咖哩蔬菜湯

食材及數量

  • 椰奶：1 碗
  • 蔬菜高湯：2 杯
  • 紅蘿蔔（切丁）：1 個
  • 馬鈴薯（切丁）：1 個
  • 洋蔥（切丁）：1 個
  • 青豆：1 杯
  • 椰子奶：1 杯
  • 綠咖哩醬：2 湯匙
  • 鹽和胡椒粉：依個人口味

做法

  1. 在中火鍋中，加入椰奶和蔬菜高湯，攪拌均勻直到混合物煮沸。

  2. 加入紅蘿蔔、馬鈴薯和洋蔥，繼續煮至蔬菜變軟。

  3. 加入青豆、椰奶和綠咖哩醬，攪拌均勻，繼續煮約5分鐘。

  4. 根據個人口味加入鹽和胡椒粉調味。

  5. 關火，盛裝起來，可加點新鮮香草或椰子片作裝飾。

2. 素食鮮蔬炒麵

食材及數量

  • 麵條：2 杯（煮熟）
  • 綠豆芽：1 杯
  • 胡蘿蔔絲：1 杯
  • 洋蔥絲：1 杯
  • 彩椒（切絲）：1 杯
  • 輕醬油：2 湯匙
  • 糖：1 湯匙
  • 米酒：1 湯匙
  • 蒜蓉：1 茶匙
  • 薑蓉：1 茶匙

做法

  1. 煮熟麵條備用。

  2. 在熱鍋中加入輕醬油、糖和米酒炒香。

  3. 加入洋蔥絲、胡蘿蔔絲和彩椒快速翻炒。

  4. 加入綠豆芽、蒜蓉和薑蓉炒勻。

  5. 最後加入煮熟的麵條，快速翻炒均勻即可。

3. 素食意大利麵沙拉

食材及數量

  • 意大利麵：2 杯（煮熟）
  • 櫻桃番茄（切半）：1 杯
  • 青瓜（切片）：1 杯
  • 甜椒（切絲）：1 杯
  • 黑橄欖（去核切片）：1/2 杯
  • 新鮮羅勒葉：1/4 杯
  • 橄欖油：2 湯匙
  • 紅酒醋：1 湯匙
  • 鹽和黑胡椒粉：依個人口味

做法

  1. 煮熟意大利麵並瀝乾水分。

  2. 將櫻桃番茄、青瓜、甜椒和黑橄欖切片備用。

  3. 在大碗中拌勻橄欖油、紅酒醋、鹽和黑胡椒粉。

  4. 加入煮熟的意大利麵和切好的蔬菜，拌勻。

  5. 最後灑上新鮮羅勒葉作為裝飾即可享用。

4. 素食黑豆墨西哥燴飯

食材及數量

  • 煮熟的白米飯：1 杯
  • 黑豆（瀝乾）：1 杯
  • 玉米粒（罐頭或冷凍）：1/2 杯
  • 切碎的番茄：1 杯
  • 切碎的生菜：1 杯
  • 切碎的紅洋蔥：1/4 杯
  • 新鮮莫札瑞拉奶酪：1/4 杯
  • 莫札瑞拉奶酪：2 湯匙
  • 番茄醬：1/4 杯
  • 橄欖油：1 湯匙
  • 辣椒粉：1 茶匙
  • 鹽和黑胡椒粉：依個人口味

做法

  1. 煮熟白米飯並備用。

  2. 在鍋中加入橄欖油和辣椒粉炒香。

  3. 加入切碎的番茄、黑豆和玉米粒炒熟。

  4. 拌入煮熟的白米飯，加入番茄醬和調味料。

  5. 最後撒上切碎的生菜和莫札瑞拉奶酪即可。

5. 素食南瓜椰奶咖哩

食材及數量

  • 南瓜：1 個
  • 椰奶：1 碗
  • 紅蘿蔔：1 個
  • 洋蔥：1 個
  • 青豆：1 杯
  • 綠咖哩醬：2 湯匙
  • 鹽和胡椒粉：依個人口味

做法

  1. 將南瓜、紅蘿蔔和洋蔥切丁備用。

  2. 在中火鍋中加入椰奶和水攪拌均勻直到混合物煮沸。

  3. 加入切丁的蔬菜，繼續煮至蔬菜變軟。

  4. 加入青豆、綠咖哩醬，攪拌均勻，繼續煮約5分鐘。

  5. 根據個人口味加入鹽和胡椒粉調味。

  6. 關火，盛裝起來，可加點新鮮香草作裝飾。
