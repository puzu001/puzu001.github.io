Html 語法架構學習

By Puzumei 23 July 2024

HTML 語法架構

介紹

HTML（HyperText Markup Language）是用於創建網頁的基本構建塊。本教學將介紹 HTML 的基本結構和常用標籤。

HTML 基本結構

一個基本的 HTML 文件由以下部分組成：

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

    <!-- 在這裡添加你的內容 -->

</body>
</html>
  • <!DOCTYPE html>：定義文件類型和版本。
  • <html>：根元素，包含整個 HTML 內容。
  • <head>：包含了文檔的元（meta）數據、鏈接到外部資源（如樣式表和腳本）和其他的元素。
  • <meta>：提供關於文檔的元數據，如字符集設置和頁面的元描述。
  • <title>：文檔的標題，顯示在瀏覽器的標題欄或標籤中。
  • <body>：包含了可見的頁面內容，如文本、圖像、超鏈接、表格等。

HTML 常用標籤

標題標籤

HTML 提供了六個級別的標題標籤，從 <h1> 到 <h6>，用於定義不同級別的標題。

<h1>這是一個一級標題</h1>
<h2>這是一個二級標題</h2>
<!-- 依此類推，直到 <h6> -->

段落標籤

用 <p> 標籤來定義段落：

<p>這是一個段落。</p>

超鏈接標籤

用 <a> 標籤來創建超鏈接：

<a href="https://example.com">點擊這裡訪問示例網站</a>

圖像標籤

用 <img> 標籤插入圖像：

<img src="image.jpg" alt="描述圖像的文字">

列表標籤

HTML 支持有序列表（<ol>）和無序列表（<ul>）：

<h3>有序列表：</h3>
<ol>
    <li>項目一</li>
    <li>項目二</li>
</ol>

<h3>無序列表：</h3>
<ul>
    <li>項目一</li>
    <li>項目二</li>
</ul>

HTML 屬性

HTML 標籤可以包含屬性，用於提供進一步的信息或設置。例如，href 屬性指定超鏈接的目標 URL。

<a href="https://example.com" title="點擊訪問示例網站">點擊這裡訪問示例網站</a>

結論

這篇教學介紹了 HTML 的基本結構、常用標籤和屬性，幫助你快速入門創建靜態網頁。HTML 是學習網頁開發的第一步，通過不斷練習和深入理解，你可以創建出豐富多彩的網頁內容。

希望這篇教學對你有所幫助！
