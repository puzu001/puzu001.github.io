### 方法三：使用特定 Markdown 編輯器的設置

- 在 Visual Studio Code 中，你可以通過安裝擴展並在設置中調整行距。

如果你是在某個特定平台（如 GitHub 或某些部落格平台）上顯示 Markdown 文件，可以查閱平台的文檔，查看是否支持自定義 CSS 或其他樣式調整方法。

要在 Visual Studio Code 中調整 Markdown 文件中程式碼區段的行距，可以通過修改相應的樣式表來實現。這裡有一個步驟指南：

打開 VS Code：確保你已經打開了你的 Markdown 文件所在的 Visual Studio Code 環境。

安裝 Markdown All in One 擴展（可選）：這個擴展提供了更多 Markdown 文件的功能，包括對程式碼區段的樣式調整支持。你可以在擴展市場中搜索 "Markdown All in One" 並安裝它。

打開使用者設定：按下 Ctrl + ,（在 Windows/Linux 上）或 Cmd + ,（在 macOS 上）打開使用者設定。

設置 Markdown 樣式：在搜索框中輸入 markdown.styles 以找到 Markdown 文件的樣式設置。

新增自訂樣式表：點擊 "在設定中編輯"（Edit in settings.json）連結，這將打開 settings.json 檔案。

新增 CSS 設定：在 settings.json 中，加入以下設定：

json
複製程式碼
"markdown.styles": [
    "file:///path/to/custom-styles.css"
]
將 file:///path/to/custom-styles.css 替換為你實際的 CSS 檔案路徑。如果你還沒有自訂的 CSS 檔案，可以創建一個新的 CSS 檔案，例如 custom-styles.css。

編輯自訂 CSS：在你的自訂 CSS 檔案中，你可以使用以下 CSS 代碼來調整程式碼區段的行距：

css
複製程式碼
/* 調整程式碼區段的行距 */
pre {
    line-height: 1.5; /* 調整為你想要的行距 */
}
這裡的 line-height 屬性可以根據你的需求調整。保存並關閉 CSS 檔案。

重新加載 VS Code：通常不需要重新啟動，但為了確保變更生效，你可以關閉並重新打開你的 Markdown 文件。

這樣就完成了 Markdown 文件中程式碼區段行距的調整。注意，這個方法可以用於調整其他 Markdown 元素的樣式，只需根據需要編輯你的自訂 CSS 檔案即可。
