Go 語法學習

By Puzumei 30 June 2024

Go 語法

Go 語言（簡稱 Golang）是一種靜態類型、編譯型的編程語言，由 Google 開發。它以其簡單、高效和並行處理能力而聞名。在這篇教學中，我們將介紹 Go 語言的基本語法，包括變量、數據類型、函數、條件語句、循環、結構體和並發處理。

1. Hello, World!

package main
import "fmt"
func main() {    
    fmt.Println("Hello, World!")    
}

以上是 Go 的一個簡單範例，展示如何打印 “Hello, World!"。每個 Go 程序都從 main 包開始執行，fmt 包提供了格式化輸出的功能。

2. 變量和常量

變量

Go 中的變量可以使用 var 關鍵字聲明：

package main
import "fmt"

func main() {
    var a int = 10
    var b = 20
    c := 30

    fmt.Println(a, b, c)
}

常量

常量使用 const 關鍵字聲明：

package main
import "fmt"
const PI = 3.14
func main() {
    fmt.Println(PI)
}

3. 數據類型

Go 支持多種數據類型，包括整數、浮點數、字符串和布爾值：

package main

import "fmt"

func main() {
    var a int = 10
    var b float64 = 3.14
    var c string = "Hello"
    var d bool = true

    fmt.Println(a, b, c, d)
}

4. 函數

Go 中的函數使用 func 關鍵字定義：

package main

import "fmt"

func add(x int, y int) int {
    return x + y
}

func main() {
    result := add(10, 20)
    fmt.Println(result)
}

5. 條件語句

Go 中的條件語句包括 if、else if 和 else：

package main

import "fmt"

func main() {
    var a int = 10

    if a < 20 {
        fmt.Println("a 小於 20")
    } else if a == 20 {
        fmt.Println("a 等於 20")
    } else {
        fmt.Println("a 大於 20")
    }
}

6. 循環

Go 支持 for 循環：

package main

import "fmt"

func main() {
    for i := 0; i < 5; i++ {
        fmt.Println(i)
    }

    // 迭代數組
    arr := [3]int{1, 2, 3}
    for index, value := range arr {
        fmt.Println(index, value)
    }
}

7. 結構體

結構體是 Go 中的自定義數據類型：

package main

import "fmt"

type Person struct {
    Name string
    Age  int
}

func main() {
    person := Person{Name: "Alice", Age: 30}
    fmt.Println(person.Name, person.Age)
}

8. 並發處理

Go 支持並發處理，使用 goroutine 和 channel：

Goroutine

package main

import (
    "fmt"
    "time"
)

func say(s string) {
    for i := 0; i < 5; i++ {
        time.Sleep(100 * time.Millisecond)
        fmt.Println(s)
    }
}

func main() {
    go say("world")
    say("hello")
}

Channel

package main

import "fmt"

func sum(s []int, c chan int) {
    sum := 0
    for _, v := range s {
        sum += v
    }
    c <- sum
}

func main() {
    s := []int{1, 2, 3, 4, 5}
    c := make(chan int)

    go sum(s, c)
    result := <-c
    fmt.Println(result)
}
